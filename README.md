# Data Visualization

A modern, interactive data visualization dashboard built with **Dash** and **Plotly**.

## Features

- Interactive data visualizations powered by Plotly
- Responsive web interface with Bootstrap components
- Efficient data processing with Pandas and NumPy
- Production-ready with Gunicorn WSGI server
- Easy deployment support (includes Procfile)

## Requirements

- dash
- dash-bootstrap-components
- pandas
- numpy
- plotly
- gunicorn

## Installation

```bash
pip install -r requirements.txt
```

## Running the Application

```bash
python data_visuvalization.py
```


## Project Structure

- `data_visuvalization.py` - Main application entry point
- `requirements.txt` - Python dependencies
